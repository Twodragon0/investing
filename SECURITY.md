# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** file a public issue.

Instead, report it privately via [GitHub Security Advisories](https://github.com/Twodragon0/investing/security/advisories/new).

We will acknowledge receipt within 3 business days.

## Supported Versions

Only the `main` branch receives security updates.

## Scope

- Direct source code in this repository
- Docker images built from this repository's Dockerfile (if applicable)

Out of scope: issues in upstream dependencies.
